
## Description
NestJs + MongoDB + Mongoose + ElasticSearch

## Installation

```bash
$ npm install
```

## Building an initial index
This will fetch all documents from the db and make elastic search index with them.
```bash
$ npm run build-index
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
